Arrays help us to see the maximum and minimum values and also the raw ideas of the distribution of ideas over the range. A much better way to represent the above data is to express it in the form of discrete or ungrouped frequency data is;

1. Discrete / Ungrouped Frequency Distributions
2. Continous / Grouped Frequency Distributions

1. Discrete / Ungrouped Frequency Distributions: When data are presented in a way that exact measurements of units are clearly indicated, it is called Discrete / Ungrouped Frequency Distribution. We count the number of times each Variables occurs or the number of times a particular value is repeated, this is called "Frequency".

To facilitate counting blocks of 5 bar //// are prepared and some space is left in between each block, finally, we count the numbers of bars to get the frequency. The presentation of data using Ungrouped Frequency Distribution is possible if the values of the variables are largely otherwise there will be no condensation.

2. Continous / Grouped Frequency Distributions: In this form of distribution variables can take any fractional values.
Hence; exact measurement of values are not possible. E.g

CLASS INTERVAL FREQUENCY
50 - 100 4
100 - 150 6
150 - 200 10
200 - 250 7
250 - 300 3
Total: 30

The following terms are to be considered when a Continous Frequency Distribution is formed.

1. Class Limit: This refers to the lowest and the highest values that must be included in a class.
The 2 boundaries of the class as known as:

i. The Lower Limit
ii. The Upper Limit

2. Class Interval: This may be defined as the size of each groupings of data, E.g 50 - 100, 100 - 150, 150 - 200, are class intervals. Each groupings begins with the lower limit of the class interval of the next succeeding class interval.

3. Class Width / Size: This is the difference between the upper and lower class limit in any given class.

4. Range: This is the difference between the largest and smallest values in a set of data or observation. it is denoted by "R". E.g 2, 4, 5, 10, 3, 8 = R = 10 - 2 = 8.

5. Class Mark / Mid Value / Mid Point: This is a central point of a class interval, it is found by adding the lower limit and upper limit of a class and dividing the sum of 2.

6. Frequency: This is the number of Observation following within a particular class interval.

7. Number of Class Intervals: The number of class intervals can be determined by sturges rules given as K = 1 + 3.322, Log ^N 10 = logarithm of a number, N = total number of observation, K is the number of Intervals. E.g

CLASS INTERVAL FREQUENCY MID POINT
50 - 100 4 75
100 - 150 12 125
150 - 200 10 175
200 - 250 4 225

K = 1 + 3.322 Log ^30 10

= 1 + 3.322 (1.4771)

= 1 + 4.9069

= 5.9069 = 6.

8. Size of a Class Interval: Size of a Class Interval is denoted as C and the formula is,
Size of a Class Interval = Range / Number of Class Interval; (C = R / K)

9. Relative Frequency: This is the Frequency of a class divided by the total frequency.

TYPES OF DATA

There are 3 types of Class Intervals;
1. Exclusive Class Interval
2. Inclusive Class Interval
3. Open End Classes

1. Exclusive Class Interval: This is a Classs Interval in which there is no difference between the upper limit of one class and the lower limit of the next class. E.g

CLASS INTERVAL FREQUENCY
50 - 100 4
100 - 150 12
150 - 200 10
200 - 250 4
Total: 30

2. Inclusive Class Interval: This is a Class Interval in which there is difference between the upper limit of one class Interval and the lower limit of the next class Interval. E.g:

CLASS INTERVAL FREQUENCY
20 - 39 5
40 - 59 10
60 - 79 2
80 - 99 4
Total: 20

3. Open End Classes: In Open End Classes a class limit is missing either at the beginning of the first Class Interval or at the end of the last Class Interval or both not specified.
